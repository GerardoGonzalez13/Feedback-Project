# Baseline Definition

Baseline behavior is defined as the aircraft’s open-loop response to a standardized upset maneuver with no autopilot or feedback controller active. Starting from a trimmed, straight-and-level condition in a clean configuration at a fixed airspeed and altitude, the aircraft is commanded through a scripted roll-in to a 60° bank angle, held briefly, and then subjected to a short rudder pulse and a small, repeatable wind disturbance to introduce roll–yaw coupling and an external perturbation. At a defined release time, all pilot control inputs are set to neutral and the aircraft is allowed to evolve freely for a fixed observation window, capturing its natural recovery characteristics (e.g., roll damping, residual oscillations, spiral tendency, and altitude loss). Baseline performance is quantified by logging bank angle and roll rate (along with heading/yaw response and altitude/vertical speed as needed) and extracting metrics such as time to return to near-wings-level, peak bank/roll-rate excursions after release, oscillation magnitude over a specified interval, and altitude loss during recovery.
