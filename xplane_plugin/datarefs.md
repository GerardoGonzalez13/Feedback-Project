# Datarefs used

Core flight dynamics:
- sim/flightmodel/position/phi            bank angle [rad]
- sim/flightmodel/position/P              roll rate [rad/s]
- sim/flightmodel/position/psi            heading/yaw [rad]
- sim/flightmodel/position/R              yaw rate [rad/s]
- sim/flightmodel/position/elevation      altitude MSL [m]
- sim/flightmodel/position/vh_ind         vertical speed indicated [m/s]
- sim/flightmodel/position/indicated_airspeed  IAS [m/s]

(We can add control surface deflections later if needed.)
